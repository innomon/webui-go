# Experimental port of Open Web UI

This repository contains an experimental port of [Open Web UI](https://github.com/open-web-ui/open-web-ui) backend from python to golang, using vibe coding.

see [backend/IMPLEMENTATION.md](backend/IMPLEMENTATION.md) for implementation details and [backend/USAGE.md](backend/USAGE.md) for usage instructions.

Its still a work in progress, and not all features may be fully functional yet.
